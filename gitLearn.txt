克隆远程库
git clone git@github.com:EricDoReMi/NoteBook.git

创建版本库
git init

添加暂存区
git add gitLearn.txt

commit版本库
git commit -m "startLearn"

显示版本库
git log --pretty=oneline

查看工作区的状态
git status

版本库返回上一个版本
git reset --hard HEAD^

版本库退回到指定版本
git reset --hard 3628164

查看版本库历史
git reflog

查看版本库和工作区的区别
git diff HEAD -- readme.txt

撤销工作区的修改---将版本库里的文档退回到工作区文档
git checkout -- readme.txt

撤销暂存区的修改
git reset HEAD readme.txt

添加远程库---远程库的名字默认为origin
git remote add origin git@github.com:EricDoReMi/gitLearn.git

列出远程库
git remote -v

推送远程库
git push -u origin restructure

创建并切换分支
git checkout -b dev

切换分支了
git checkout dev

查看分支了
git branch

在一个分支上merge其他的分支
git merge dev

保留dev分支的合并
git merge --no-ff -m "merge with no-ff" dev

删除单个分支
git branch -d dev

保存工作区
git stash

=====标签就是给版本号一个别名============

创建一个标签---默认是最后一个版本
git tag v1.0

给具体版本打标签
git tag v0.9 6224937

查看具体标签
git tag

创建带说明的标签
git tag -a v0.1 -m "version 0.1 released" 3628164

查看标签说明文字
git show v0.1

删除一个标签
git tag -d v0.1

推送某个标签
git push origin v1.0

推送全部标签
git push origin --tags
